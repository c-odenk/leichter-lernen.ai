# 📚 leichter-lernen.ai – Deine KI-gestützte Lernplattform

<br>

leichter-lernen.ai ist eine webbasierte Lernplattform, die moderne KI-Technologien nutzt, um Studierende, Schüler und andere Lernende beim effizienten Lernen zu unterstützen. Durch die Analyse eigener Dokumente erstellt die Plattform Zusammenfassungen, Lernkarten und bietet einen persönlichen KI-Tutor zur Vertiefung des Verständnisses.
<br>
<br>
Aktuell zu finden unter: https://leichter-lernen.netlify.app/
<br>
<br>
## 🚀 Executive Summary
Die Plattform richtet sich an alle, die ihren Lernprozess smarter gestalten wollen. Durch den Upload von Vorlesungsskripten oder Lernunterlagen analysiert die KI die Inhalte automatisch und liefert:
<br>
<ul>
  <li>strukturierte Zusammenfassungen</li>
  <li>digitale Karteikarten</li>
  <li>Multiple-Choice-Tests</li>
  <li>einen interaktiven KI-Tutor</li>
</ul>
So sparen Nutzer Zeit, lernen gezielter und können ihren Wissensstand effektiv überprüfen.
<br>
<br>

## 🛠️ Tech Stack
Die Anwendung basiert auf dem MEVN-Stack: Das Frontend wurde mit Vue.js umgesetzt und kommuniziert über eine REST-API mit einem Express.js-/Node.js-Backend. Die persistente Datenhaltung erfolgt in einer MongoDB-Datenbank, gehostet über MongoDB Atlas. Der KI-Workflow wird über eine externe Flowise-Instanz realisiert, die auf Render gehostet ist. Die Kommunikation erfolgt über eine API, über die die Analyse der Dokumente sowie die Generierung von Zusammenfassungen, Lernkarten und Tests abgewickelt wird.
